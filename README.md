# 🚀 Software Engineering Portfolio



A minimalist, high-performance personal portfolio website built to showcase software engineering projects, academic research, and UI/UX design work. Designed with a premium dark-mode aesthetic and optimized for internship applications.

## ✨ Key Features



* **Modern Tech Stack:** Built using Next.js App Router, React, TypeScript, and Tailwind CSS.

* **State-Driven Interactions:** Uses React `useState` for dynamic project modals and interactive UI/UX galleries.

* **Custom Carousel:** Features a custom CSS-transform-powered horizontal slider for academic research papers.

* **Premium UI/UX:** Dark-mode design with vibrant accent colors, glassmorphism navigation, and smooth scrolling.

* **Fully Responsive:** Mobile-first architecture with a custom hamburger menu for smaller screens.

```text
│

├── Projects.tsx

├── Research.tsx

└── Design.tsx



public/

└── projects/

```



### Component Overview



* `app/page.tsx` – Main single-page layout integrating all sections.

* `components/Navbar.tsx` – Responsive sticky navigation with scroll detection.

* `components/Hero.tsx` – Landing section with dynamic background effects and CTAs.

* `components/Projects.tsx` – Project showcase with grid-to-modal interactions.

* `components/Research.tsx` – Research section with a custom horizontal slider.

* `components/Design.tsx` – UI/UX gallery and freelance design showcase.

* `public/projects/` – Optimized image assets.

## 💻 Getting Started



### 1. Clone the Repository



```bash

git clone https://github.com/Dulshan-Rajeewa/my-portfolio.git

```



### 2. Navigate to the Project Directory



```bash

cd my-portfolio

```



### 3. Install Dependencies



```bash

npm install

```



### 4. Start the Development Server



```bash

npm run dev

```



Open **http://localhost:3000** in your browser to view the application.
