# 🚀 Software Engineering Portfolio

A minimalist, high-performance personal portfolio website built to showcase software engineering projects, academic research, and UI/UX design work. Designed with a premium dark-mode aesthetic and optimized for internship applications.

## ✨ Key Features

* **Modern Tech Stack:** Built using the Next.js App Router, React, and styled entirely with Tailwind CSS.
* **State-Driven Interactions:** Utilizes React `useState` for dynamic project modal overlays and an interactive UI/UX design gallery.
* **Custom Carousel:** Features a custom-built, CSS-transform-powered horizontal slider for academic research papers.
* **Premium UI/UX:** Implements a strict dark-mode color palette (`#0a0a0a` & `#121212`) with vibrant accent highlights (`#ff6b00`), glassmorphism navigation, and smooth scroll behaviors.
* **Fully Responsive:** Mobile-first design ensures perfect rendering across all devices, including a custom mobile hamburger menu.

## 📂 Project Structure

The application is structured into modular React components for maintainability and scalability:

* `app/page.tsx` - The main single-page layout integrating all sections.
* `components/Navbar.tsx` - Responsive, sticky header with scroll detection.
* `components/Hero.tsx` - Landing section with dynamic background glows and CTAs.
* `components/Projects.tsx` - Software engineering portfolio featuring a grid-to-modal architecture.
* `components/Research.tsx` - R&D section utilizing a custom horizontal sliding track.
* `components/Design.tsx` - Interactive UI/UX gallery and freelance brand identity summary.
* `public/projects/` - Local directory for highly optimized Next.js Image assets.
